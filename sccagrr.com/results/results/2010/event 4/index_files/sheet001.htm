All event data for the day is entered on the "Today" sheet! Do NOT insert rows!!!
If you need to enter someone new, overwrite someone not present.
Do not enter any data below line 100. 		When the event is complete click on the "Results" button and view results on the "by Class", "Street Tire", and "Ladies" tabs. The "Novice" and "Bumped" tabs are not used.
Column Data requried Further description of data required Reason
Number Enter Car number
Driver Enter Driver
Vehicle Enter Car Description
GRR Member Enter Member Y for GRR member only. Used for season points calcs
Class Entered Enter class (normal class for this car assuming no bumping
Class for this event Bumped class or Novice This column will be filled automatically, but can be overwritten when bumping.
Enter N for novice
Enter B for bumped		 Used to group single cars to form a class and/or a novice class.
Tires Street or Race S for street is all that is required. Blank assumes race tires
Gender M or F F is all that is required, Blank assumes male
Novice Y Blank assumes not novice
Click the "Sort by" button to sort competitors by number on the "Today" sheet only. (This button may be clicked at any time to sort the competitors.)
Click the Results button to copy data from the "Today" sheet to "By Class", "Street Tire", "Ladies" and "Novice" sheets.  You can now sort by Class to check for cars that should be moved to the Bumped class which is designated by placeing a "B" in the "Class for this event column. Class changes MUST be entered on the "Today" sheet to be effective. The results button may be clicked at any time. Data on the other sheets will be overwritten.
